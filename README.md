# 财务工具箱

基于 Vue 3 的财务工具集合，提供发票处理等实用工具。

## 工具列表

### 发票工具

#### 发票重命名
批量处理发票文件，智能提取信息并重命名。

#### 发票解析
批量处理发票文件，解析金额、税额、价税合计、税率等信息，并支持以Excel、JSON导出

## 安装

```
# 安装依赖
npm install
```

## 运行

```
# 开发模式
npm run dev

# 构建生产版本
npm run build

# 预览生产版本
npm run preview
```
